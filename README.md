# 📝 Advanced To-Do List

A modern and feature-rich To-Do List application designed to help users organize tasks efficiently, manage priorities, and boost productivity.

## ✨ Features

- ➕ Add new tasks
- ✏️ Edit existing tasks
- ❌ Delete tasks
- ✅ Mark tasks as completed
- 🔍 Search and filter tasks
- 🚩 Set task priorities
- 📅 Add due dates and deadlines
- 📂 Organize tasks into categories
- 💾 Persistent data storage
- 📱 Responsive design for desktop and mobile
